Maze Controls

In the maze the character can be controlled using the arrow keys or W A S D. Using two diagonal keys simultaneously allows movement at 45 degree angles.

Goal

The goal of the game is to complete a full carbon cycle. This is done by answering all the questions correctly and completing each maze. The exit in each maze is marked with a red flag.
